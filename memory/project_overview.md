SaaS de chatbots para reserva de citas. Stack: backend + frontend separados.

**Core flow:** Usuario se registra → 7 días de prueba gratuita → pasa a plan de pago.

**Planes:**
- Plan básico (pago): 1 chatbot
- Plan Pro (pago): hasta 2 chatbots

**Chatbot:** El usuario crea un chatbot, comparte el link con sus clientes, y los clientes registran citas de forma autónoma.

**Admin:** Solo un usuario admin (Alberto, albertojprada10@gmail.com) con capacidad de dar/revocar accesos y gestión general.

**Why:** Producto propio del usuario, en construcción activa.
**How to apply:** Tener en cuenta los límites por plan al implementar features de chatbot/citas.
